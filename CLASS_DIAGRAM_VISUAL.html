🔗 Jenis Relasi

Inheritance (--|>): Pewarisan antar class (generalisasi/spesialisasi)

Association (-->): Asosiasi antar class dengan hubungan M:1 atau 1:1

📊 Cardinality

M:1: Many-to-One (banyak ke satu)

1:1: One-to-One (satu ke satu)

Extends: Pewarisan/inheritance

📌 Catatan

Garis-garis digambar tegak lurus tanpa belok untuk kejelasan visual

Semua relasi sesuai dengan ERD original yang Anda buat

FK (Foreign Key) direpresentasikan sebagai asosiasi dalam diagram
